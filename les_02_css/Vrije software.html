Open source

Open source Open source is een term die eind jaren negentig is bedacht om gebruik te maken vrije software door bedrijven te stimuleren. Door sterk de nadruk te legge op rechten en het vrijgeven van zoveel mogelijk broncode hadden veel bedrijven de indruk dat zij gebruik konden maken van vrije software
